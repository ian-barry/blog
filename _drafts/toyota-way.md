#### Glossary

| Word     | Meaning                             |
| -------- | ----------------------------------- |
| TPS      | Thinking (Toyota) Production System |
| Kaizen   | Continuous Impovement               |
| Muda     | Wastes                              |
| Jidoka   | Built-in quality                    |
| Kanban   | Signal of some kind                 |
| Heijunka | Leveling out the work schedule      |

The **kanban** is an organized system of inventory buffers.
Kanban is something you strive to get rid of, not to be proud of.
